Google Sheet columns to Docs

Google Workspace

  • Key highlights

  • Served as a solo developer on the project
  • Developed a Google Apps Script to convert Google Sheets to Google Docs
  • Script supports arbitrary number of sheet columns
  • Automated Google Doc formatting is planned for future engagement

Summary

As a solo developer on the project, I created a simple Google Apps Script that converts Google Sheet columns to a set of Google Docs. The script uses the header row as the name of each file. All rows in each column are populated one after the other into the body of every Google Doc.

Read sheet columns

Client users populate a single sheet with project information. Every project is listed in a single column.

Instead of sharing the sheet, the client needs to share the information in separate Google Docs.

Automated file creation

The script creates a Google Doc for each sheet column, using the top row of every column as the doc name.

The script then moves each file to a designated target Google Drive folder.

Fully populated document

The final result of the script is a Google Doc that contains all the content of the sheet column.

The script uses the first sheet column as doc section headers, and incorporates the user's content underneath each header.
