Chrome extension for site scraping

Google Apps Script

SUMMARY

The client needed a Chrome extension, which his team would use to mark their website pages for scraping. A Google Apps Script accepts the extension's page requests, scrapes the web page, and saves the data in a Google Doc based on a template.

Chrome extension

A Chrome extension is used to identify web site pages for scraping. When users click the extension, it sends the web page URL to a Google Apps Script, which crawls the page, and retrieves specific information.

Custom data scraping

The script can pull information from the HTML head or body. A set of functions enables the script to locate HTML elements by tag name, class name, or id.

The script can pull out different aspects of the desired HTML elements, including their inner text, HTML, value, and attributes.

Customizable template

A Google Doc template is used for the production of documents based on website data.

The client identifies the location of each website element that should be included in the file.

Final document

Once the script scrapes the web page, it duplicates the template. It then replaces each placeholder with the actual site data.

The script renames the file and stores it in a specific Google Drive folder for additional processing.
