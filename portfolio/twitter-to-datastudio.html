Pull Twitter data into Google Data Studio automatically

Google Apps Script

SUMMARY

This Google Apps Script queries the Twitter API, pulling data about tweets and followers into a Google Sheet, which is synched with a Google Data Studio report.

Twitter Data API

A Google Apps Script is configured to call the Twitter data API and pull data about tweets and followers of a specific Twitter account.

For this to work, the Twitter account must be a Twitter Developer account with access keys.

Google Sheet Updates

Every week, the Google Apps Script is triggered to pull last week's Twitter data, and append it to a Google Sheet.

The script is flexible enough to handle changes in column order inside the sheet.

Google Data Studio

The Google Sheet serves as a data source for a Google Data Studio report.

Users can refresh the data and get the latest Twitter information.
