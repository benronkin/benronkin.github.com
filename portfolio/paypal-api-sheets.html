PayPal API into Google Sheets

Google Apps Script

SUMMARY

This Google Apps Script pulls PayPal transactions into Google Sheets using the PayPal transactions API. Clients can run the script to pull transactions automatically every month, or pull data for custom date ranges, such as year-to-day, previous quarter, and more.

Set up a PayPal app

The client quickly sets up a PayPal app on developer.paypal.com and enables transaction searches.

The app provides a web address to pull the PayPal transactions.

Automated script

The Google Apps Script uses the PayPal app credentials to access the client data based on the client's desired reporting period.

Google Sheet report

The script pulls all of the PayPal transactions for the desired date range. From each transaction the script pulls the data elements most relevant to the client.

The script also includes calculated metrics that are added to the dataset as well. The script saves the transactions to the client's Google Sheet.
