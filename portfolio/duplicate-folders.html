Duplicate files and folders automatically

Google Apps Script

SUMMARY

Duplicating folders in Google Drive is currently a manual process in which yoy create folders one at a time. This Google Apps Script automates the process. Users select the folder they want to copy and its target folder. The web app duplicates all files and sub-folders. Users can run this process automatically on a schedule.

Easy folder selection

Users leverage a Google Drive picker to select the folder they wish to copy, and where to save it.

The web app can remember folder selection for future use.

Configurable process

Users can select to duplicate only sub-folders or to duplicate files as well.

The web app an be run on demand or configured to run automatically on schedule.

Deep folder duplication

The web app duplicates deeply-nested folders and files, regardless of how deep folders are. It visits every sub-folder and checks for nested sub-folders within.
