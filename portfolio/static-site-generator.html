Summary

In this project, I delivered a static website generator to a MarTech vendor who needed to showcase its solutions on demo websites for different industry verticals. The generator is fully customizable: The client generates category and product feeds for retail, financial services, and other verticals. The client modifies page template for each vertical. The generator then uses the feeds and the templates to generate the static sites.

Page types for different use cases

The resulting sites are equipped with different site sections and page types to help demo your MarTech/AdTech technology across different workflows and use cases.

For instance, pages can render product rating and reviews in cart pages, based on information supplied in product feeds.

Static sites for interactive demos

The generated sites use vanilla Javascript to simulate user interactions, such as searching, browsing, filtering, and more.

Sales engineers who deliver sales demo can interact with the pages and showcase how their MarTech solution responds to user events.

Easy site creation and hosting

Creating new static site is easy: The client needs only to create fake product metadata and images. The client then runs a Gulp builder in Node.js to produce the end result.

The generator uses Bootstrap, so changing site themes is as is as dropping in a theme file.

The static sites include standard HTML, CSS and Javascript files that are easy to host on GitHub Pages or other static site hosting services.
