Consolidate multiple Google Calendars for Upwork

Google Apps Script

SUMMARY

I needed to sync my Upwork appointment scheduling with my multiple calendars without giving Upwork direct access to my calendars' events. I created a new Google account with a calendar that uses a Google Apps Script to aggregate all of my busy time slots from all my calendars.

Automate calendar updates

The script runs every five minutes automatically. It accesses all of my personal Google Calendars and fetches all of the events scheduled for up to a month in the future.

The script has access to a calendar in a different Google account of mine. The script creates events on that calendar using the start and end times of my events. No information about the events themselves is stored in that calendar; only their start and end times.

Automated availability sync

I connected my new Google Calendar with Upwork scheduling. My availability on Upwork updates automatically as the underlying Google Calendar gets updated.

In addition, I can manually add events to the calendar when I want to limit my Upwork availability without cluttering my personal calendars.
