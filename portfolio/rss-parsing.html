RSS feed parsing

Google Apps Script

  • Key highlights

  • Parses any number of RSS feeds
  • Accepts feed URLs in Google Sheet
  • Outputs feeds' metadata
  • Outputs feeds' entries
  • Manual and time-based trigger

Summary

The project needed to routinely parse a number of RSS feeds and store key information about each feed item. I created a Google Apps Script that parses an arbitrary list of feed URLs, and updates a Google Sheet with metadata and entries of each feed. The script can be run directly from the sheet menu or automatically using a timer.

Configurable feed list

The client enters the list of RSS feed urls in a Google Sheet. The list can chance at any time.

Once the list is entered, the client starts the script using a custom sheet menu item.

Feed metadata

The script updates the metadata sheet with information on each feed.

Each sheet row contains information on one feed.

Feed items

The script creates a running list of all the feeds' items. Users can configure the list of attributes they wish to pull for each item.

The script maintains previous pulls and uses the item's GUID to ensure that each item is pulled only once. The script uses custom logic to fall back on channel-level data when item-level data isn't present.

Automated parsing

In addition to running manually, the script also runs automatically via a timer-based trigger. The trigger is easy to update.
