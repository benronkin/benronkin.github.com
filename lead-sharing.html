Data management app

Google Cloud

Summary

As a solo developer on the project, I've been designing, developing, releasing, and maintaining a secured Node.js Express web application on Google Cloud Platform. The app enables ecosystem partners to receive and pursue sales leads, and communicate their progress with their partner manager.

Authentication and authorization

The app enables partner managers to register their partners. Each manager sees only their partners' data. Each partner sees only their data.

Authentication is implemented by Google Cloud Platform Identity-Aware Proxy. All requests are signed with JWT.

Authorization is handled by a roles and permissions system that I built from scratch. Every request is screened for operation permission and data access permission. Users are assigned to roles, and roles have permissions.

Reduced server load

The app stores only pertinent facts in the database and push them to the client. All derived attributes are calculated client-side in real-time.

Filtering, for example, is also done client-side, using jQuery to hide and show data rows as needed.

Data persistence

As required by the project's client, the app uses Google Firestore NoSQL database. New leads are fed directly into the database using ETL workflow that I designed and developed using Google Apps Script.

RELATED CASE STUDIES

The project required developing a Node.js app on Google Cloud Platform that processes campaign registrations from Google Sheets. The app reads the sheet via Google Sheets API, groups the registrations by campaign ID, and creates a CSV file for each campaign.

The file then gets uploaded to Google Cloud Storage, where the app sets access permission on the file. Finally, the app emails a download link to the CSV file.

The client needed to share sales leads with channel partners in a secured way. I implemented a solution that ingests sales leads and assigned partners, builds CSV files on Google Cloud Storage, and routinely emails partners links to the CSV files.

Partners must authenticate into their Google account to download the file.
