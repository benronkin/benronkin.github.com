What are the requirements for using this add-on? +
You need to have a Gmail account or a Google Workspace account. The add-on is installed inside Google Sheets. You will need to create two Google Drive folders for the layouts and the resulting banner images. You will need to place a Google Slide file in the layouts folder. The add-on will guide you through the setup process.
How do I use this add-on? +

You enter banner information in the tracking sheet that the add-on creates for you. Each row in the sheet represents one banner file. Select the layout that you want for the banner, enter your desired banner file name, and enter the text strings that will replace the placeholders. Click on Create Banners.

The add-on will create a banner image for any row that does not contain an existing file URL. If you want to recreate the image then simply delete its file URL.
