Google Sheet to Asana via Zapier

Google Workspace

Key Highlights

This Google Apps Script pulls new entries from a Google Sheet in real-time. It uses a Zapier webhook to insert new to-do tasks into an Asana project. The payload includes a mix of standard and custom Asana fields. The script updates the sheet with Zapier IDs to eliminate duplicate inserts.

Read sheet entries

A Google Sheet is routinely updated with new asset requests.

The team needs the information pushed into Asana, so that they can effectively manage the project requests.

Automated sheet processing

An Installable Trigger watches the sheet for changes. When new rows are inserted into the sheet, the trigger executes a Google Apps Script.

The script scans the sheet for new entries (i.e, rows without a Zapier ID). It packages the new entries and pushes the payload to a Zapier webhook.

Zapier webhook

The final result of the script is a Google Doc that contains all the content of the sheet column.

The script uses the first sheet column as doc section headers, and incorporates the user's content underneath each header.

Real-time Asana inserts

The Zapier webhook inserts new to-do tasks into Asana.

The team then manages the requests directly in Asana.
