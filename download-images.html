Download images automatically to Google Drive

Google Workspace

SUMMARY

This Google Apps Script solution downloads images from the web into a Google Drive folder. Users can provide a Google Sheet with a list of image URLs. Or they can specify a web page address that the solution will crawl and fetch its images.

Automated image download

The script can run automatically by observing a Google Drive folder, waiting for a new sheet with a list of image URLs.

When a sheet is dropped in the folder, the script will download the images.

Manual control

Alternatively, users can leverage a Google Apps Script web app, where they can specify where to pull the images from, and where to save them.

Users can either point to a web address of a page that contains the images or to a sheet that contains the image URLs.

Finally, users specify the Google Drive folder where the images should be save to.

Downloaded images

The solution will store all the images encountered in a Google Drive folder.

This script provides information about the download: count of images, and names, sizes, and URLs of the images.
