RSS feed parsing

Google Apps

  • Key highlights

  • Parses any number of RSS feeds
  • Accepts feed URLs in Google Sheet
  • Outputs feeds' metadata
  • Outputs feeds' entries
  • Manual and time-based trigger

Summary

The client needed to routinely parse a number of RSS feeds. I created a Google Apps Script that reads the client's feed list, and provides metadata and entries of each feed. Client can run the script manually or automatically using a timer.

Configurable feed list

The client enters the list of RSS feed urls in a Google Sheet. The list can chance at any time.

Once the list is entered, the client starts the script using a custom sheet menu item.

Feed metadata

The script updates the metadata sheet with information on each feed.

Each sheet row contains information on one feed.

Feed entries

The script creates a running list of all the feeds' entries. It's easy to group and pivot the table by the feed title.

The script iterates through the list of categories of each feed entry.

Automated parsing

In addition to running manually, the script also runs automatically via a timer-based trigger. The trigger is easy to update.
