Scrape website pages

Google Workspace

Key Highlights

This Google Apps Script accepts one or more website URLs to scrape. It fetches the HTML code of each URL and then extracts different data elements from each page, based on business needs, and stores the information in a Google Sheet.

Website crawling

A Google Apps Script is configured to visit website URLs and pull their HTML code. The list of URLs is configurable.

Custom data scraping

The script can pull information from the HTML head or body. A set of functions enables the script to locate HTML elements by tag name, class name, or id.

The script can pull out different aspects of the desired HTML elements, including their inner text, HTML, value, and attributes.

Save the scraped data

The script stores the website content that it extracts in a Google Sheet.

Here, the script visited the website home page and extracted the URLs of all the portolio pages that are listed on the home page.
